Comment sélectionner les "Agenda" où sont notés mes appels ?
Dans l'application Android : Cafe-crm>Menu (en bas à gauche)>List Calendar: Selectioné un ou des calendriers
Comment importer un agenda dans mon Android ?
Dans le téléphone android : Setting>Accounts>Add Account phone-log
Quel type d'agenda est susceptible de convenir ?
L'application a été testé avec les "Google Calendar". Elle devrait fonctionner avec d'autres calendriers (Yahoo, MS ...) Nous faire savoir le résultat.
Comment visualiser sur internet mes appels archivés dans un Agenda?
1 - Il faut synchroniser les agendas (Cela peut être fait de façon automatique ou manuelle sur le téléphone android: Setting>account>"MyCount@gmail.com">accounts>synchronize
2 - L'agenda une fois synchronisé, je peux voir mes appels sur Internet en allant sur la page https://www.google.com/calendar/ si j'utilise un agenda de google.

Comment importer des événements (Commandes, Facturations, Livraisons etc )?
Menu (Bouton Gauche)>Import>Imports Events
Sur cet écran,soit vous importer un fichier que vous avez préalablement stocké (Reçu par email et téléchargé par exemple)
Soit si vous connaissez l'url du fichier, vous tapez l'url et importer le fichier.
Quel doit être le format des fichiers importés d'évènement?
Ce doit être des fichiers texte ou csv, sans formatage, avec en debut de chaque ligne la date au format dd/mm/yyyy (par exemple 12/02:2014)
